# Shoot the fruit

- Open your terminal and clone this repository

    First make sure you have a ssh key if you dont have go to my dotfiles repository and follow the instructions

    ```
    git clone git@github.com:ayushkpai/shoot-the-fruit.git
    ```

- Next install python and pygame zero

    Also documented in dotfiles

- To run the files

    ```
    uv run pgzrun Code.py
    ```
